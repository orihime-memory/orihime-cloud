# 織姫フロント接続パッチ

## 使い方
1. `index.html` と `app.js` を同じフォルダに置く
2. FastAPI を起動する
3. この HTML をブラウザで開く
4. API URL が `http://127.0.0.1:8000` ならそのままでOK

## 画像について
今のフロントは画像ファイルを `/api/chat` へ送れる。
ただし、バックエンド側で保存・解析処理を足すまでは、
「送信できるUI」までが整っている状態。

## 夢機能について
今の `/api/dream` は手動保存。
24:00 自動起動にするなら、バックエンド側で APScheduler を使って
毎日 24:00 に diary / logs を圧縮するジョブを追加すると良い。
